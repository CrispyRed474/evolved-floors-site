ARCHITECTURE
─────────────────────────────────────────────────────────────────────────────────

[Browser Form] → [Cloudflare Worker] → [GHL API] → [SMS + CRM]
  (HTML/JS)        (serverless)         (contacts/opps)  (Twilio)

Data Flow:
  1. User fills form → Step 1-6
  2. Form submits → POST /submit
  3. Worker creates contact + sends SMS
  4. User receives SMS with 6-digit code
  5. User enters code → POST /verify
  6. Worker verifies code + creates opportunity
  7. Contact tagged "web-verified" triggers GHL workflow
  8. Workflow sends estimate email
